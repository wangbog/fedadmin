Welcome to FedAdmin

A web based management tool for NREN identity federation administrators to operate federations, managing Identity Providers (IdPs) and Service Providers (SPs), and exchanging metadata with eduGAIN metadata service.

Deployer Note

This is a default landing page for the open-source FedAdmin project. The FedAdmin deployer should customize this page for the actual federation by editing ./app/templates/main/index.html.
