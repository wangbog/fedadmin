## 0.1.0 - Initial public release

- Initial FedAdmin web application for managing federation organizations, administrator users, IdPs, and SPs.
- Added beta and production metadata workflows with entity application and approval states.
- Added federation metadata generation, metadata transformation, and eduGAIN metadata import/update support.
- Added Docker-based development and production deployment documentation.
- Added documentation for administrator workflows, scheduled tasks, storage layout, backlog, known issues, and glossary.
- Added MIT license, security policy, contribution guidelines, and changelog.
- Added initial GitHub Actions test workflow and helper unit tests.
- Added development dependency file for local tests.
- Hardened URL validation, custom action redirects, and metadata file cleanup behavior.
